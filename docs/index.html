How to Create Art

Pattern Bloom

Use Pattern Bloom to auto-fill the canvas with mathematically generated drops. Every click rotates to a new pattern style with varied drop sizes and palette combinations.

Interaction Modes

Drops: paint with your selected palette color. Random Palette: random radius and palette color. Random Full: random radius and any RGB color. Tine: carve marbling lines. Press T to toggle Tine quickly.

Tine Controls

Adjust Strength for displacement force and Sharpness for tine influence radius. Drag across the canvas in Tine mode for smooth carved waves.

Palette Workflow

Choose a Palette Set, then choose a color swatch for the next drop. Swatches select color only; they do not switch the full palette set.
